# 🦁 LikeLion 2025 백엔드 트랙 학습 아카이브

> 멋쟁이사자처럼 대학 2025년도 백엔드 트랙 활동 중 수행한 **주차별 과제와 학습 기록**을 체계적으로 정리한 포트폴리오입니다.

---

## 🎯 학습 목표

AI 엔지니어를 목표로 하는 과정에서, **AI 모델을 실제 서비스에 통합하기 위해서는 백엔드 개발의 기초와 시스템 전체 아키텍처에 대한 이해가 필수**라고 판단했습니다.

이를 위해 LikeLion 백엔드 트랙에 참여하여 다음과 같은 역량을 체계적으로 습득하고 있습니다:

- **Java & Spring Framework**를 활용한 백엔드 개발 역량
- **REST API, 데이터베이스, 인증** 등 핵심 백엔드 기술 이해
- **팀 협업과 과제 해결**을 통한 실무적 문제 해결 능력

## 🎯 Learning Objectives

As someone aspiring to become an AI Engineer, I recognized that **understanding backend development fundamentals and system architecture is essential for integrating AI models into real-world services**.

Through participation in LikeLion's Backend Track, I am systematically acquiring the following capabilities:

- **Java & Spring Framework** backend development skills
- Core backend technologies: **REST APIs, databases, authentication**
- **Team collaboration and problem-solving** abilities through practical assignments
